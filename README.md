# 🏠 Inmuebles del Litoral

Repositorio del proyecto **Inmuebles del Litoral**, desarrollado como parte de un ejercicio de control de versiones utilizando **Git** y **GitHub**.

## 📋 Descripción

Este repositorio implementa una estrategia de trabajo con múltiples ramas para organizar el desarrollo, las pruebas y la versión estable del proyecto.

## 🌿 Estructura de ramas

- **produccion** → Contiene la versión estable del proyecto.
- **Desarrollo** → Rama utilizada para el desarrollo de nuevas funcionalidades.
- **pruebas** → Rama destinada a realizar pruebas antes de integrar los cambios a producción.

## 🛠 Tecnologías

- Git
- GitHub

## 🚀 Flujo de trabajo

1. Desarrollo de funcionalidades en la rama **Desarrollo**.
2. Verificación y validación en la rama **pruebas**.
3. Integración de los cambios aprobados en la rama **produccion**.
