# 📖 Descripción

**Inmuebles del Litoral** es un proyecto desarrollado siguiendo una metodología de control de versiones basada en **Git Flow**, implementando una estructura de ramas para garantizar un desarrollo organizado, seguro y escalable.

Este repositorio representa un entorno profesional donde cada modificación atraviesa distintas etapas antes de llegar a producción, asegurando la calidad del software mediante procesos de desarrollo y validación.

---

# 🎯 Objetivos del Proyecto

✔ Implementar buenas prácticas de Git.

✔ Gestionar un repositorio remoto mediante GitHub.

✔ Trabajar con ramas independientes.

✔ Simular un ambiente real de desarrollo colaborativo.

✔ Mantener un historial limpio de versiones.

✔ Aplicar integración progresiva del código.

# 🏗 Arquitectura del Repositorio

```text
                    GitHub
                       │
        ┌──────────────┼──────────────┐
        │              │              │
        ▼              ▼              ▼
  produccion      Desarrollo      pruebas
       │               │              │
       └───────────────┼──────────────┘
                       ▼
              Versiones del Proyecto
```

---

# 🌿 Estrategia de Ramas

| Rama | Estado | Descripción |
|:------|:------:|:------------|
| 🟢 **produccion** | ✔ Estable | Contiene la versión oficial del proyecto. |
| 🟡 **Desarrollo** | 🚧 Desarrollo | Implementación de nuevas funcionalidades. |
| 🔵 **pruebas** | 🧪 Testing | Validación antes de integrar cambios. |
